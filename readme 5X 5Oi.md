### *Описание игры*
```
5 X 5 O - отличная игра,
созданная по мотивам классической версии крестики-нолики,
вот только здесь мы имеем игровое поле размером 5 на 5.
```
### *Как играть?*
```
 Вы будете играть друг против друга.
 Для победы в данной игре нужно составить целый ряд из 5-ти крестиков или ноликов,
 при этом ряд может быть как вертикальный, так и горизонтальный или диагональный. 
 Оба игрока ходят по очереди и устанавливают свой значок в любую из свободных 
 клеточек.

 Чтобы победить в игре нужно первым составить целый ряд из 5-ти своих фигур
 и не дать сделать этого вашему сопернику.
 Для начала сыграйте несколько игр, чтобы посмотреть как играет ваш соперник
 и научиться у него разным приемам. После этого можно применять 
 эти же приемы против вашего соперника и возможно таким образом вам
 удастся выиграть. 
```
### Необходимые программы и модули 

Для коректного запуска файла `igra 5 X 5 O.py` необходимо установить 
Python версии не ниже 3.10, также VS Code версии не ниже 1.79.

Данная игра была протестирована на работоспособность на следующей платформе:
OS Windows 10 Pro 22H2, Python 3.11.2, VS Code 1.79.2.


### Запуск игры

Файл `igra 5 X 5 O.py` запускается с помощью команды RUN в VS Code:

```
RUN igra 5 X 5 O.py
```

### Основные функции работы приложения
```
Дан первый список на 25 значений, который будет заполнен цифрами от 1 до 25. 
Пользователю будет достаточно ввести символ от 1 до 25,
что бы мы смогли понять куда он сделал ход. После
того как ход сделан, цифру мы заменим на символ нолика или крестика. 

Дан второй список с информацией о выигрышных комбинациях. 

Функция будет выводить содержимое нашего списка на экран.

Также одна функция будет рисовать на поле крестик или нолик, в зависимости от того, что в
нее передали. Вставлять переданный элемент мы будем по индексу функцией index.
После каждого ходы мы должны проверять — не победил ли кто то из игроков. 
Для этого переберем все победные линии из второго списка выйгрышных комбинаций. 

Основной цикл работает до момента пока кто нибудь не выиграет.
Игроки будут по очереди делать ходы, мы будем проверять каждый раз не победил ли кто
и как только кто то побеждает —
выходим из цикла и завершаем программу.
```
